* __Disponible dans :__ [toutes les templates](https://github.com/Etana/template#templates)

```html
{L_LOGIN_LOGOUT}
```

## Description[*](https://fa-tvars.appspot.com/var/L_LOGIN_LOGOUT)
Chaîne de caractère.
