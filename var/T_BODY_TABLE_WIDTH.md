# Variable T_BODY_TABLE_WIDTH

##Disponibilité
Cette variable est utilisable directement.

```html
{T_BODY_TABLE_WIDTH}
```

##Description[*](https://fa-tvars.appspot.com/var/T_BODY_TABLE_WIDTH)
