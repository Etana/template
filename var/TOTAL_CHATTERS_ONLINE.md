# Variable d'affichage
```
TOTAL_CHATTERS_ONLINE
```


##Description[*](https://fa-tvars.appspot.com/var/TOTAL_CHATTERS_ONLINE)
Chaîne de texte indiquant le nombre d'utilisateurs connectés sur la chatbox.

* __Contenu en français :__ `Il y a actuellement X utilisateur(s) sur la ChatBox`

##Disponibilité
Cette variable est utilisable directement.

```html
{TOTAL_CHATTERS_ONLINE}
```
