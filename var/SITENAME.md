# SITENAME


Cette variable est utilisable directement.

```html
{SITENAME}
```

##Description[*](https://fa-tvars.appspot.com/var/SITENAME)
Contient le titre de votre forum défini dans votre `PA > Général > Forum | Configuration > Nom du site`.
