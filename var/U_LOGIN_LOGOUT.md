# U_LOGIN_LOGOUT
* __Type :__ variable d'affichage
* __Disponible dans :__ [toutes les templates](https://github.com/Etana/template#templates)
* __Utilisation :__

```html
{U_LOGIN_LOGOUT}
```

## Description[*](https://fa-tvars.appspot.com/var/U_LOGIN_LOGOUT)
Adresse url absolue vers : 

* pour un membre connecté => une page de déconnexion directe : par exemple `/login?logout=1&tid=4488721b0bee072288240e2015539676&key=cf8668`
* pour un invité => une page de connexion : `/login?connexion`
