# L_INDEX


Cette variable est utilisable directement.

```html
{L_INDEX}
```

##Description[*](https://fa-tvars.appspot.com/var/L_INDEX)
Chaîne de texte contenant le titre de votre forum défini dans `PA > Général > Forum | Configuration > Nom du site`.
