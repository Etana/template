## Description[*](https://fa-tvars.appspot.com/var/JS_DIR)
Adresse URL du dossier dans lesquel se trouvent les fichiers script généraux de forumactif.

* __Exemple de remplacement :__ `https://illiweb.com/rs3/40/frm/`
