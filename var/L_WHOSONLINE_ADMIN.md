# L_WHOSONLINE_ADMIN
* __Type :__ variable d'affichage
* __Disponible dans :__ [toutes les templates](https://github.com/Etana/template#templates)
* __Utilisation :__

```html
{L_WHOSONLINE_ADMIN}
```

## Description[*](https://fa-tvars.appspot.com/var/L_WHOSONLINE_ADMIN)
Code html avec le texte `Administrateur` mis de la couleur indiquée dans le champ `Couleur Police [Administrateur]` dans [`Index`](http://votre-forum.appspot.com/#/admin/,&part=themes,&mode=colors&sub=logos) > [`Panneau d'administration`](http://votre-forum.appspot.com/admin/#&part=themes,&mode=colors&sub=logos) > [`Affichage`](http://votre-forum.appspot.com/admin/?part=themes#&mode=colors&sub=logos) > [`Images et Couleurs | Couleurs`](http://votre-forum.appspot.com/admin/?mode=colors&part=themes&sub=logos).

* __Exemple de remplacement français :__ 

```html
<span style="color:#aa0000" class="coloradmin">Administrateur</span>
```
