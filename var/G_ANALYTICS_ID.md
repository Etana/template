# G_ANALYTICS_ID


Cette variable est utilisable directement.

```html
{G_ANALYTICS_ID}
```

##Description[*](https://fa-tvars.appspot.com/var/G_ANALYTICS_ID)
Affiche la valeur indiquée dans `PA > Général > Statistiques | Google Analytics > Identifiant de compte`.
